# 運営者画面一覧

運営者は最高レベルの管理権限を持ち、事業所管理、ケアマネージャー管理、システム設定を行います。

**合計画面数**: 17画面

## 優先度について

- **0**: 最高優先度（認証・基本管理機能）
- **1**: 高優先度（日常業務機能）
- **2**: 中優先度（補助機能）
- **3**: 低優先度（研修・マニュアル機能）
