# CareBase-admin 画面一覧

CareBase-adminアプリケーションの全画面一覧です。3つのユーザー役割に分けて整理されています。

## ユーザー役割別画面一覧

- **[運営者（17画面）](./運営者.md)** - 最高レベルの管理権限、事業所管理、ケアマネージャー管理
- **[事業所管理者（57画面）](./事業所管理者.md)** - 日常的な介護業務管理、最も多くの機能を利用
- **[ケアマネージャー（8画面）](./ケアマネージャー.md)** - ケアプラン管理に特化した機能

## 優先度について

- **0**: 最高優先度（認証・基本管理機能）
- **1**: 高優先度（日常業務機能）
- **2**: 中優先度（補助機能）
- **3**: 低優先度（研修・マニュアル機能）
