# Tamanomi-admin 画面一覧

Tamanomi-adminアプリケーションの全画面一覧です。加盟店管理システムとして再構成されています。

## 機能別画面一覧

- **加盟店管理** - 加盟店の登録、編集、一覧表示
- **ユーザー管理** - ユーザーアカウントの管理
- **管理アカウント** - システム管理者アカウントの管理
- **お問い合わせ** - ユーザーからのお問い合わせ管理

## 主な機能

| 機能 | 説明 |
|------|------|
| 加盟店管理 | 加盟店の登録、編集、削除、一覧表示 |
| ユーザー管理 | ユーザーアカウントの作成、編集、権限管理 |
| 管理アカウント | システム管理者の管理 |
| お問い合わせ | サポート・問い合わせの管理 |
