# 事業所管理者画面一覧

事業所管理者は日常的な介護業務の管理を行い、最も多くの機能を利用します。

**合計画面数**: 57画面

## 優先度について

- **0**: 最高優先度（認証・基本管理機能）
- **1**: 高優先度（日常業務機能）
- **2**: 中優先度（補助機能）
- **3**: 低優先度（研修・マニュアル機能）
