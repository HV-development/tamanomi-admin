# ケアマネージャー画面一覧

ケアマネージャーはケアプラン管理に特化した機能を利用します。

**合計画面数**: 8画面

## 優先度について

- **0**: 最高優先度（認証・基本管理機能）
- **1**: 高優先度（日常業務機能）
- **2**: 中優先度（補助機能）
- **3**: 低優先度（研修・マニュアル機能）

---

## 認証関連（4画面）

| No. | 画面名                 | 概要                                                                                                                | 主要機能               | 優先度 |
| --- | ---------------------- | ------------------------------------------------------------------------------------------------------------------- | ---------------------- | ------ |
| 1   | ログイン               | メールアドレスとパスワードでログインする<br>認証が成功した場合TOPページへ遷移し、失敗した場合エラーメッセージを返す | ログイン               | 0      |
| 2   | パスワードリマインダー | パスワードを忘れた際にログイン情報のメールアドレスを入力する                                                        | パスワードリマインダー | 0      |
| 3   | パスワードリセット     | メールで送信されたリンクから遷移してパスワードをリセットする<br>有効期限切れの場合、有効期限切れの画面へ遷移する    | パスワードリセット     | 0      |
| 4   | 事業所切替             | ケアマネージャーに紐づく事業所から操作する対象を選択する<br>選択した事業所に紐づく利用者とケアプランのみ参照できる  | -                      | 0      |

## ケアプラン管理（3画面）

| No. | 画面名         | 概要                                                                                                                                                          | 主要機能                         | 優先度 | 備考                                      |
