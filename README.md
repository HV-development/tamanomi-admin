# Tamanomi Admin

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)](https://carebase-admin.vercel.app)

## 概要

Tamanomi Adminは加盟店管理システムの管理者向けアプリケーションです。加盟店管理、ユーザー管理、管理アカウント、お問い合わせなどの包括的な機能を提供します。

## デプロイ

アプリケーションは以下のURLで公開されています：

**[https://tamanomi-admin.vercel.app](https://tamanomi-admin.vercel.app)**
